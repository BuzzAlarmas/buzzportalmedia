# buzzportalmedia

How to get permalink.

* Copy image path for example: logo_no_bg_w.png
* Then paste this infront of it: https://raw.githubusercontent.com/BuzzAlarmas/buzzportalmedia/main/
* Final permalink should be something like: https://raw.githubusercontent.com/BuzzAlarmas/buzzportalmedia/main/logo_no_bg_w.png
